# Azure project Spotify

Projet d'ingénierie des données Spotify complet de A à Z avec à des outils comme :
- Azure Data Factory, 
- Azure SQL DB, 
- Azure Databricks, 
  - Unity Catalog, 
  - Delta Live Tables, 
  - Spark Streaming, 
  - PySpark, 
  - Databricks Asset Bundles, 
- GitHub, 
- Modélisation des données dimensionnelles 
- Dimensions à évolution lente (SCD)
